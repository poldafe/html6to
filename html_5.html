HTML 5

¿Qué es html 5 y cómo lo utilizo?

HTML 5 se trata de una actualización de HTML que incluye mas etiquetas y mejoras para el diseño Web.

Para crear una página en HTML 5 debemos empezar nuestro código de la siguiente manera:

<!DOCTYPE html>

DOCTYPE como su nombre lo indica hace referencia  al tipo de documento, con esto los navegadores web detectarán que se trata de html5, al menos que la versión de tu navegador Web sea muy antigua.

Luego se integra la etiqueta <HEAD></HEAD> donde conviene indicar que la codificación del documento es UTF-8, un formato de codificación que permite que el navegador acepte caracteres especiales y presente correctamente palabras con tildes, con la letra ñ, etc.
