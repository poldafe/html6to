Recordando HTML

¿Recuerdas lo básico de HTML?

Estructura básica en HTML

Títulos en páginas Web:

Para escribir un título en una página Web coloca dentro del <body> la etiqueta <h1> seguido del texto que deseas como título, finalmente cierra la etiqueta de la siguiente manera: </h1>

Ejemplo:

<h1> Títulos en páginas Web </h1>

 Puedes utilizar títulos más pequeños utilizando las etiquetas <h2>, <h3>, <h4>, <h5> o <h6>

Ejemplo:

<h1>Título principal</h1>  Resultado: Título principal

<h2>Título secundario</h2> Resultado: Título secundario

<h3>Título más pequeño</h3> Resultado: Título más pequeño

...

Párrafos en página Web

Los párrafos se escriben utilizando la etiqueta <p> dentro de la etiqueta <body>

Ejemplo:

<p>Es muy fácil escribir párrafos en páginas web.</p>

<p>Este es un segundo párrafo de mi página Web.</p>

Enlaces en páginas web

Para colocar enlaces a otros archivos o páginas Web utiliza la etiqueta <a> y coloca la ruta o enlace en su atributo "href"

Ejemplo:

<a href="https://www.6toInformatica.com"> Pulsa este texto para ir al sitio Web de 6to de Informática </a>
